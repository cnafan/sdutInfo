## 简介  
采用python3的requests库添加header即可抓取。

##  
由于学校的学生信息是存在学校自己的服务器上，而且它默认每个学生的帐号是自己的学号，密码是身份证后六位。只要没有改过密码就可以进去查看自己的个人信息以及成绩记录。都说证件照才是一个人的真实颜值，于是就写了个爬虫，爬到了自己班级所有人的信息。不得不说，有了动力效率会大大提升
